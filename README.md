# Portfolio Website

This repository contains the source code for my personal portfolio website, featuring detailed case studies, an infinite AI assets scroll page, and more.

## Development

The project is built with HTML, CSS, and vanilla JavaScript and bundled with Vite.

### Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Structure

- `index.html`: Main portfolio homepage.
- `styles/`: Global and component-specific stylesheets.
- `js/`: Application logic and scripts.
- `scripts/`: Assorted data extraction tools and utilities.
- `.html` files: Individual case study and auxiliary pages.
